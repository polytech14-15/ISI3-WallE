# Robocup Rescue

Lors du lancement de l’application, une nouvelle fenêtre apparaît. On y retrouve 3 parties :
- Le panel de simulation qui représente la carte de la simulation avec le graph et les différents robots et incendies.
- Le panel de configuration pour créer le graphe.
- Le panel pour ajouter des éléments pendant la simulation.


Dans un premier temps, il faut soit importer soit créer un graph. Pour créer un graph, il suffit de cliquer sur la carte pour ajouter des noeuds. Pour ajouter des arrêtes, il faut sélectionner un noeud de départ en cliquant dessus : il se colore en bleu. Puis il faut cliquer sur le noeud d'arrivée pour créer l’arrête.  Il est possible de choisir le type d'arête ou de noeud en sélectionnant le type désiré parmi les listes déroulantes.
Pour revenir en arrière, il suffit de cliquer sur le bouton “Back”.

Il est possible d’exporter le graphe en appuyant sur le bouton “Export” pour l’utiliser ultérieurement. Pour importer un graphe il suffit de cliquer sur importer et de sélectionner un fichier de type OSM XML conforme.


Après avoir créé un graph, on peut ajouter des robots de plusieurs types différents ainsi que des feux sur certains noeuds. En choisissant le type d’algorithme et en cliquant ensuite sur “Play”, la simulation démarre et les robots font tout ce qu’il peuvent pour aller éteindre les feux.
Il est possible d’arrêter la simulation en cliquant sur “Stop”.
Pour réinitialiser le graph, il suffit de cliquer sur “Reset”.

En cas d’un besoin d’aide, un menu d’aide et également présent.
